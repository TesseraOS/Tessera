Tessera's MCP server exposes <McpToolCount /> tools. This catalog is generated by
spawning the real server over stdio and recording its `tools/list` answer — names,
descriptions, and input schemas below are **what the server actually reports**, not a
maintained copy.

To connect an agent to these tools, see the [per-agent setup guides](/docs/agents).
For the concepts behind the important ones: the
[Context Compiler](/docs/concepts/context-compiler) (`compile_context`, `explain`),
[effect-links](/docs/concepts/effect-links) (`get_effects`, `assert_effect`),
[memory](/docs/concepts/memory) (`capture_memory`), and
[retrieval](/docs/concepts/retrieval) (`search`).

Results are returned both as human-readable content and as machine-readable
`structuredContent`; responses are deliberately token-lean (the verbose path is
`explain`, by design).
